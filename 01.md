## githib 코딩 파일 첫 업로드
- 내 프로필 > repositories 클릭 > new 클릭 > Repository name 코딩 파일 제목 작성 ( 영어 ) > Description 파일 설명 작성 ( 자유 ) > 퍼블릭 or 프라이빗은 유료 > create repository 클릭
git init - VS code 좌측 패널에서 git 아이콘 클릭 > initialize repository 클릭 > 업로드 할 파일만 + 버튼 > staged changes 로 이동 > 수정이 되면 다시 +
message 입력 칸에 어떤 것이 변경 되었는지 입력하고, commit 버튼 클릭 ( branch 는 생략 ) 
터미널에 아래 코드 순서대로 붙여넣기 / git remote add origin https://github.com/appletieo3o-png/gitstudy-260701.git
git push -u origin main
로그인 필요하면 로그인 하고 밑에 초록 버튼 누르고, github 들어가서 새로고침 하면 완료

## github 코딩 파일 수정 후 업로드
- 업로드 할 파일 + staged changes 이동 > message 입력 칸에 어떤 것이 변경 되었는지 입력하고 commit 버튼 클릭 > sync chages 버튼 클릭하면 내 컴퓨터에서 github 으로 수정된 코드 전송
